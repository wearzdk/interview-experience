1. **RAG与大模型**：RAG（检索增强生成）流程：文档切片→向量化（Embedding）→存向量数据库→检索时将query向量化→TopK语义检索→将相关文档拼入prompt→LLM生成。优化：混合检索（语义+关键词）、重排序Rerank、查询改写、上下文压缩。评估：召回率（relevant docs retrieved/total relevant）、精确率、Answer相关性。
