### 《面试题目》

1. 介绍 Transformer 的基本原理和注意力机制，并解释扩散模型。
2. 你如何理解 Agent 与 Workflow？
3. 你了解 Lovart 吗？你认为它属于 Agent 还是 Workflow，判断依据是什么？
4. 当前 AI 产品的二次编辑能力较弱，如果由你设计，会如何改进？
5. 广告业务已有素材生成、投放等多种工具能力，应做一体化产品，还是拆成多个产品？为什么？
6. 以广告素材生成为例，如何证明工具或产品的商业价值？生成内容的 Reward 如何衡量？
7. 如果主 Agent 安排子 Agent 执行广告任务，而子 Agent 判断需要额外生成素材，这项新任务应如何流转和审批？

---

### 《参考解析》

1. **Agent与Workflow**：Workflow 的步骤和分支主要由人预先定义，Agent 会依据目标与环境动态规划并调用工具。实际产品常是二者结合：稳定主流程用工作流，不确定环节交给 Agent。
2. **二次编辑设计**：把生成结果拆成可定位对象，让用户通过选区、自然语言和参数进行局部修改；系统应保留版本、编辑历史和约束，避免每次修改都全量重生成。
3. **一体化与多产品**：依据用户任务是否连续、角色是否一致、数据能否共享和组织销售方式判断。广告从素材到投放若目标一致，可用统一工作台承载，同时保持模块边界。
4. **商业价值评估**：同时看素材生产时间和成本、采用率、审核通过率，以及投放后的点击、转化和增量收益；不能只用生成质量分替代真实业务结果。
5. **多Agent流转**：子 Agent 应返回新增任务的原因、预算、依赖和预期收益，由主 Agent 判断是否纳入计划；涉及预算或发布的动作必须经过明确权限和人工确认，并保留审计记录。
