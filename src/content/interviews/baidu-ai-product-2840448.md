### 《面试题目》

1. 你有 B 端产品经验吗？
2. 过去哪段实习最令你印象深刻？
3. 你过去的工作更偏运营，是否真正推动过产品上线？
4. 你如何看待国内外大模型的发展和差异？
5. 你通常如何做竞品分析？
6. 你了解 RAG 吗？
7. 当前 AI 产品有哪些商业化模式？
8. Cursor 与 Claude Code 有什么区别？
9. Claude Code 与 OpenClaw 有什么区别和联系？
10. 你认为 OpenClaw 为什么会流行？
11. 你如何从零开始设计并落地一款软件？
12. 为什么想做产品经理？

### 《参考解析》

1. **B端经验**：重点说明客户角色、采购与使用者差异、复杂权限、交付流程和可量化业务价值。即使没有正式 B 端经历，也可用相近项目说明对这些约束的理解。
2. **RAG**：基本链路是文档解析与切分、向量化、检索、重排、上下文组装和生成。评估不能只看回答流畅度，还要看召回、事实准确、引用可追溯和时延成本。
3. **AI商业化**：常见模式包括订阅、按量调用、席位费、增值功能和按结果收费。选择依据是价值可衡量程度、成本结构、使用频率和客户预算习惯。
4. **AI编程工具比较**：Cursor 以编辑器内交互和代码上下文为核心，Claude Code 更偏终端中的代理式开发；OpenClaw 类通用 Agent 更强调跨工具执行，使用边界和风险也更广。
5. **产品上线能力**：应讲清需求验证、范围裁剪、原型和技术评审、开发验收、灰度发布、指标监控及复盘，突出自己对结果负责的环节。
