### 《面试题目》

1. 举一个例子说明 Java 类和对象的关系
2. 什么是多态
3. 会用画图软件画类图吗
4. 会用前端 Vue 吗？Vue2 和 Vue3 有什么区别？
5. 用过哪些前端框架

### 《参考解析》

1. **类和对象的关系**：类是对一类事物共同属性和行为的抽象描述（模板/蓝图），对象是根据类创建出来的具体实例，拥有类中定义的属性和方法的具体值；举例来说，"汽车"是一个类，定义了颜色、品牌、行驶等属性和方法，而"一辆红色的比亚迪汉"就是这个类的一个具体对象。
2. **多态的含义**：多态指同一个方法调用，根据调用对象的实际类型不同而表现出不同的行为，Java 中通过继承+方法重写（编译时看父类引用类型、运行时看实际对象类型来动态绑定调用哪个方法）实现，典型体现是父类引用指向不同子类对象时调用同名方法产生不同的执行结果。
3. **Vue2 与 Vue3 的主要区别**：响应式原理上 Vue2 用 `Object.defineProperty`（无法监听新增/删除属性、数组变化需特殊处理），Vue3 改用 `Proxy`（能完整监听对象和数组的变化）；API 风格上 Vue3 引入组合式 API（`Composition API`，逻辑复用更灵活），同时具备更小的打包体积和更好的 TypeScript 支持。
