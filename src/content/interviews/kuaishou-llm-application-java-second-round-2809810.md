### 《参考解析》

1. **Spring AI Alibaba 与 RAG 优化**：重点突出 Alibaba 在接入阿里云百炼平台和模型生态上的便利性。RAG 优化可从预处理（切片、清洗）、检索（混合检索、重排序 Re-rank）、后处理（Prompt 增强）三个层面回答。
2. **向量数据库索引**：主流算法包括 HNSW（图索引，速度快）、IVF（倒排索引，平衡性好），面试时需说明为何针对业务选型。
3. **Java 集合体系**：Collection 接口下分 List（有序可重复）、Set（无序不可重复）、Queue（队列）。HashMap 继承自 AbstractMap 并实现 Map 接口，底层是数组+链表+红黑树。
4. **设计模式应用**：JDK 中观察者模式（java.util.Observer）、迭代器模式（Iterator）；Spring 中工厂模式（BeanFactory）、代理模式（AOP）。单例建议补充双重校验锁（DCL）及枚举式单例。
5. **并查集（Union-Find）**：用于处理不相交集合的合并与查询问题，核心是 `find`（查找根节点，带路径压缩）和 `union`（合并两个集合，按秩优化）操作。
