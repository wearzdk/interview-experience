### 《面试题目》
1. 自我介绍
2. 拷打两段实习经历
3. 拷打 Agent 项目
4. 平时用哪些 AI coding 工具
5. 反问环节：总共几面、业务内容、面试评价

---

### 《参考解析》
1. **实习经历拷打**：重点准备 STAR 法则。明确自己在项目中的角色、解决的核心难点（如高并发、数据一致性）、以及通过该经历学到了什么技术沉淀。
2. **Agent 项目拷打**：面试官通常关注 Agent 的感知、决策与执行能力。需清晰阐述 Prompt 工程、工具调用（Function Calling）流程以及 Agent 的记忆机制（如 RAG 或向量数据库应用）。
3. **AI Coding 工具应用**：列举如 Cursor、GitHub Copilot、Kite 等。重点在于说明如何利用这些工具提升开发效率，例如自动生成单元测试、快速理解开源代码库或辅助 debug。
4. **反问建议**：通过询问“业务部门主要使用的技术栈”或“当前团队面临的核心挑战”，可以展示你对该岗位的真实兴趣和思考深度。
