### 参考解析
1. **AI 工具使用经验**：重点在于体现你如何通过 AI 提升开发效率（如代码生成、Debug、重构）。强调你不是简单使用，而是结合 IDE 插件解决具体业务痛点。
2. **OpenClaw/Skill 的理解**：在 Agent 开发框架中，Skill（技能）通常指 Agent 可调用的具体能力或 API 接口。执行原理一般涉及：Prompt 分析意图 -> 匹配 Skill 列表 -> 自动填充参数 -> 执行并获取返回结果。
3. **人工智能基础**：即使是非 AI 岗位，了解基础知识如机器学习（线性回归、逻辑回归、决策树）、深度学习（Transformer、Attention 机制）及大模型基础（Prompt Engineering、RAG）会增加竞争力。
4. **业务迁移大模型**：这是当前企业级大模型的典型方向。建议储备 RAG（检索增强生成）、向量数据库使用、模型微调（Fine-tuning）及 Prompt 优化等相关知识。
