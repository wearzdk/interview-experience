## 参考解析

> **注意：本次面试几乎无技术题目，以下为背景说明与求职建议。**

### 关于外包岗位面试风格

部分银行外包岗位面试侧重候选人稳定性与态度，而非纯技术考核。建议求职者提前了解外包公司背景，明确用人单位是外包方还是银行本身。

### 关于面试后的附加任务

被要求完成涵盖前端、后端、容器的 AI 文档，远超 Java 后端岗位范畴。遇到此类情况建议：
1. 明确任务是否属于正式评估环节；
2. 确认任务范围与岗位 JD 是否匹配；
3. 如范围严重不符，可礼貌拒绝或向 HR 确认岗位定位。

### 关于银行外包 Java 后端常见考点（备考建议）

虽然本次面试未涉及，但同类岗位通常会考察：
- Java 基础：集合、多线程、JVM；
- Spring Boot / Spring Cloud 框架使用；
- 数据库 SQL 优化与事务管理；
- Redis 缓存、消息队列（如 Kafka/RabbitMQ）基本概念；
- Docker/K8s 容器基础（外包岗越来越常见）。
