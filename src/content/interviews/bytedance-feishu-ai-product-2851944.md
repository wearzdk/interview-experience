### 《面试题目》

1. 工作中搭建了 Agent，为什么不采用 Workflow 或“提示词加大模型”的方案？
2. Agent 在这个场景中具体完成了什么不可替代的工作？
3. 使用 Agent 有哪些弊端？
4. 你使用过飞书多维表格吗？具体用过哪些模块？

### 《参考解析》

1. **方案边界**：单轮且无外部依赖的任务适合直接调用大模型；路径固定、步骤明确的任务适合 Workflow；只有输入和执行路径不可预测、需要动态规划与选工具时，Agent 才更有价值。
2. **选型表达**：先说明业务的不确定性，再举出 Agent 根据中间结果动态调整计划或选择工具的真实例子，同时承认其稳定性和成本代价。
3. **Agent 弊端**：常见问题包括输出不稳定、链路难调试、多次调用成本高和循环风险。可通过最大步数、结构化日志、关键节点确认和失败终止机制降低风险。
4. **多维表格准备**：应能结合实际场景介绍表结构、公式、自动化、仪表盘和 AI 字段，而不只是回答“用过”。
