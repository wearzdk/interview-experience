每天自动采集牛客、脉脉等平台最新面经，覆盖字节跳动、腾讯、美团、快手、滴滴、阿里巴巴、携程等公司，涵盖 Java、前端、Go、算法等热门岗位。

## 本地开发

```bash
bun install   # 安装依赖
bun dev       # 启动开发服务器 localhost:4321
bun build     # 构建
```
